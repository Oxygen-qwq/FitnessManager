# FitnessManager
ПО для систематизации услуг фитнесс-центра

Описание проекта Данный проект представляет собой учебное Java-приложение, предназначенное для автоматизации и централизованного управления бизнес-процессами фитнесс-центра. Основная идея проекта — упростить работу администраторов и тренеров за счёт объединения учёта клиентов, абонементов, расписания тренировок и отчётности в единой информационной системе. Проект разработан в рамках учебной работы и демонстрирует применение объектно-ориентированного программирования, проектирования реляционных баз данных и создания многослойной архитектуры приложения с графическим интерфейсом.
Целью проекта является создание прототипа приложения, которое: автоматизирует регистрацию и учёт клиентов фитнесс-центра; упрощает управление абонементами и услугами; оптимизирует планирование групповых и персональных тренировок; обеспечивает контроль посещаемости и сроков действия абонементов; формирует аналитические отчёты по загруженности и доходам; закрепляет практические навыки проектирования и разработки полноценных информационных систем на языке Java.

Основной функционал: регистрация клиентов с медицинскими анкетами; каталог услуг и гибкая система абонементов; календарь расписания групповых занятий; модуль бронирования персональных тренировок; управление персоналом (тренеры, администраторы); система уведомлений об истечении абонементов; генерация финансовых и статистических отчётов; роль-based доступ для разных категорий пользователей.
Используемые технологии Java 17; JavaFX для графического интерфейса; Hibernate/JPA для работы с базой данных; PostgreSQL 14; Maven для сборки проекта; объектно-ориентированное программирование; паттерны проектирования (MVC, DAO); Git и GitHub для контроля версий.

Установка и запуск Склонируйте репозиторий с помощью команды git clone https://github.com/ваш-логин/fitness-center-manager.git Откройте проект в IntelliJ IDEA как Maven-проект Убедитесь, что установлен сборщик Maven и JDK 17 Настройте подключение к базе данных в файле src/main/resources/application.properties Выполните инициализацию базы данных скриптом из папки db/schema.sql Запустите приложение через класс ru.fitness.MainApplication.java

Структура репозитория Основной код расположен в src/main/java/ru/fitness/ с подпакетами model (сущности), repository (доступ к данным), service (бизнес-логика), controller (управление интерфейсом); ресурсы приложения находятся в src/main/resources/views (FXML-макеты), styles (оформление), а также конфигурационные файлы и скрипты БД в корне resources и папке db.
Дальнейшие улучшения В дальнейшем проект может быть улучшен. Планируется добавление мобильного клиента, интеграция с платежными системами, расширенная аналитика с визуализацией данных и поддержка онлайн-записи клиентов через веб-интерфейс.

Проект создан в учебных целях и не предназначен для коммерческого использования.
